## üêõ Troubleshooting

### Logo Still Not Showing After 7 Days?

1. **Check Logo File**
   - Make sure `/public/images/shake-chilli-logo.png` exists
   - Recommended size: 512x512px or 1200x1200px
   - Format: PNG with transparent background
   - File size: < 100KB

2. **Validate Structured Data**
   - Use Google's Structured Data Testing Tool
   - Ensure no errors in JSON-LD markup
   - Verify logo URL is absolute (not relative)

3. **Check robots.txt**
   - Ensure `/images/` is not blocked
   - Logo file should be crawlable

4. **Request Indexing**
   - Use Google Search Console
   - Request indexing for homepage
   - Check mobile usability

5. **Verify Domain**
   - Ensure domain is verified in Search Console
   - Check for any manual actions or penalties
